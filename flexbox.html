Aparato multifuncional

Entrena musculos de las piernas asi como del pecho.

Banco press

Este banco plano que cuenta con un soporte para colocar la barra, es una de las máquinas más básicas que encontrarás en el gimnasio, y que si bien puede servir de apoyo para la realización de muchos ejercicios, se utiliza frecuente para realizar press de banca para trabajar pectorales.

Paralelas

Sirve para realizar abdominales en paralelas, por ejemplo, llevando las rodillas al pecho, o bien, podemos trabajar tríceps y pectorales realizando fondos o dippings en esta máquina.

Dorsaleras

Posee un asiento y soporte para los muslos, tiene un sistema de poleas y una barra que debemos sujetar para jalar de ella y trabajar los músculos de la espalda. En este aparato se pueden realizar jalones trasnuca, para ejercitar entre otros músculos, dorsales.
